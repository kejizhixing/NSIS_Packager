# NSIS可视化打包工具 功能详解
该工具是一款面向Windows平台的NSIS（Nullsoft Scriptable Install System）可视化打包辅助软件，旨在简化NSIS安装包的制作流程，降低普通用户使用NSIS的技术门槛，核心功能与相关说明如下：

## 一、核心功能
### 1. 可视化操作界面，替代手动编写NSIS脚本
无需掌握NSIS脚本语法，通过图形化界面完成安装包制作的全流程配置，包括源程序目录选择、主程序指定、安装包参数设置等，大幅降低使用成本。

### 2. 智能信息识别与自动填充
- 自动扫描选定的程序目录，识别目录内的EXE文件并列出供用户选择作为主程序；
- 自动读取主程序（EXE/DLL）的版本信息（如产品名称、版本号、出版商、版权信息等），并填充到配置项中；若未读取到版本信息，可自动从文件名提取基础信息并补充默认值；
- 自动检测程序目录内的ICO图标文件，支持一键选用，也可手动选择自定义图标或NSIS内置图标库中的图标。

### 3. 灵活的安装包参数配置
- **应用信息配置**：支持自定义应用名称、版本号、出版商、版权信息、官方网站、安装目录名称等核心参数；
- **输出配置**：可自定义安装包输出路径与文件名，默认将输出文件保存到当前工作目录的output文件夹，避免临时目录丢失问题；
- **许可证协议配置**：支持编辑自定义软件许可协议，也可使用工具默认的许可协议模板，协议内容会嵌入到安装包中，安装时展示给用户。

### 4. NSIS环境适配与便捷操作
- 自动检测系统中已安装的NSIS目录（支持常见安装路径，如C:\Program Files (x86)\NSIS等），也可手动指定NSIS安装目录；
- 提供“生成脚本”“编译安装”“一键打包”等快捷功能：生成符合配置的NSIS脚本文件，调用NSIS的makensis.exe编译生成安装包，一键打包可完成脚本生成+编译全流程。

### 5. 进度可视化与日志记录
- 编译过程中实时展示进度条，清晰显示打包进度；
- 全程记录操作日志（如目录选择、信息识别、脚本生成、编译结果等），便于排查问题。

### 6. 配置保存与加载
支持将常用的配置（如出版商、版权信息、NSIS目录等）保存为JSON配置文件，后续可一键加载，减少重复配置操作。

## 三、其他实用特性
- 兼容中文路径：完美支持包含中文的程序目录、文件名、图标路径等，解决传统NSIS脚本编写中中文乱码问题；
- 临时文件管理：自动创建独立临时目录存放脚本、许可证等中间文件，避免文件混乱；
- 多语言版本信息读取：支持读取中文简体、英文等多语言版本的程序信息，适配不同地区的软件打包需求；
- 窗口与消息框自定义：支持加载自定义窗口图标，消息提示框带有专属图标，提升操作体验。

该工具核心价值在于将NSIS的命令行/脚本操作转化为可视化交互，保留NSIS免费、灵活的优势，同时降低使用门槛，适合需要快速制作Windows安装包但不熟悉NSIS脚本的用户。
